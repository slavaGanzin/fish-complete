# gencomplete

[![Build Status][travis-badge]][travis-link]
[![Slack Room][slack-badge]][slack-link]

Autogenerate completions for command using --help

## Install

With [fisherman]

```
fisher slavaGanzin/fish-gencomplete
```

## Usage

```fish
gencomplete commandname [--help-key]
```
